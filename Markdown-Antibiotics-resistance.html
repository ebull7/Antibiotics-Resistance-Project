Antibiotic Resistance in Clinical Salmonella (USA, 2015–2025)

Elijah

2026-04-25

Introduction

Antimicrobial resistance (AMR) in Salmonella enterica represents a growing public health threat in the United States. Fluoroquinolones such as ciprofloxacin are among the most critical treatments for severe salmonellosis — resistance to these drugs significantly limits clinical options. This analysis examines resistance trends across five antibiotic classes in 57,297 clinical Salmonella isolates collected between 2015 and 2025, with a focus on identifying which serovars are driving observed changes.

Data source: NCBI Pathogen Detection Database. Analysis restricted to clinical (human) isolates from US submissions with complete AMR genotype data.

Results

Overall Resistance Trends

trends <- df |>
  group_by(year) |>
  summarize(
    betalactam    = mean(res_betalactam) * 100,
    tetracycline  = mean(res_tetracycline) * 100,
    fluoroquinol  = mean(res_fluoroquinol) * 100,
    sulfonamide   = mean(res_sulfonamide) * 100,
    chloramphenic = mean(res_chloramphenic) * 100
  )

trends_long <- trends |>
  pivot_longer(cols = -year, names_to = "resistance", values_to = "percent")

ggplot(trends_long, aes(x = year, y = percent, color = resistance)) +
  geom_line(linewidth = 1.2) +
  geom_point(size = 2.5) +
  annotate("rect", xmin = 2021.5, xmax = 2025.5, ymin = 0, ymax = 27,
           alpha = 0.08, fill = "red") +
  annotate("text", x = 2022, y = 26.5, label = "Post-2022 surge",
           size = 3.2, color = "red3", hjust = 0) +
  scale_color_manual(
    values = c(
      "betalactam"    = "#E63946",
      "tetracycline"  = "#F4A261",
      "fluoroquinol"  = "#2A9D8F",
      "sulfonamide"   = "#457B9D",
      "chloramphenic" = "#6A4C93"
    ),
    labels = c(
      "betalactam"    = "Beta-lactams (blaTEM/CMY)",
      "tetracycline"  = "Tetracyclines (tet)",
      "fluoroquinol"  = "Fluoroquinolones (gyrA/qnr)",
      "sulfonamide"   = "Sulfonamides (sul1/2)",
      "chloramphenic" = "Chloramphenicol (floR)"
    )
  ) +
  scale_x_continuous(breaks = 2015:2025) +
  labs(
    title = "Antibiotic Resistance in Clinical Salmonella (USA, 2015–2025)",
    subtitle = "% of isolates carrying resistance genes by antibiotic class",
    x = "Year", y = "% Resistant Isolates", color = "Resistance Class",
    caption = "Source: NCBI Pathogen Detection Database | n = 57,297 clinical isolates"
  ) +
  theme_minimal(base_size = 13) +
  theme(plot.title = element_text(face = "bold"),
        panel.grid.minor = element_blank())

Across all five antibiotic classes, resistance rates remained broadly stable between 2015 and 2021. Fluoroquinolone resistance is the notable exception, rising sharply from approximately 12% in 2021 to 25% by 2025 — a trend investigated further in Section 3.

California showed notably elevated fluoroquinolone resistance relative to other states. This likely reflects a combination of population density, high international travel volume, and intensive agricultural antibiotic use in the Central Valley. It should be noted that geographic coverage is uneven — most states are gray due to insufficient NCBI submissions, reflecting lab participation patterns rather than true absence of resistance.

Serovar Breakdown

serovar_trends <- df |>
  filter(serovar %in% c("Enteritidis", "Typhimurium", "Newport",
                         "Infantis", "Typhi")) |>
  group_by(year, serovar) |>
  summarize(
    pct_fluoroquinol = mean(res_fluoroquinol) * 100,
    n = n(), .groups = "drop"
  ) |>
  filter(n >= 30)

ggplot(serovar_trends, aes(x = year, y = pct_fluoroquinol, color = serovar)) +
  geom_line(linewidth = 1.2) +
  geom_point(size = 2.5) +
  facet_wrap(~ serovar, ncol = 3, scales = "fixed") +
  scale_x_continuous(breaks = c(2015, 2018, 2021, 2025)) +
  labs(
    title = "Fluoroquinolone Resistance by Salmonella Serovar (2015–2025)",
    subtitle = "% of isolates carrying gyrA/qnr resistance genes",
    x = "Year", y = "% Resistant Isolates",
    caption = "Source: NCBI Pathogen Detection Database | minimum n=30 isolates per year/serovar"
  ) +
  theme_minimal(base_size = 12) +
  theme(plot.title = element_text(face = "bold"),
        panel.grid.minor = element_blank(),
        legend.position = "none")

S. Enteritidis is the primary driver of the overall fluoroquinolone surge, rising from ~20% resistance in 2015 to ~65% by 2025. S. Typhi maintains chronically high resistance (~80%) throughout the study period — a separate and serious clinical concern given fluoroquinolones are a primary treatment for typhoid fever. S. Infantis, Newport, and Typhimurium remain comparatively stable and low.

Resistance Projection: S. Enteritidis to 2030

enteritidis_trend <- serovar_trends |>
  filter(serovar == "Enteritidis")

model <- lm(pct_fluoroquinol ~ year, data = enteritidis_trend)

future_years  <- data.frame(year = 2025:2030)
predictions   <- predict(model, newdata = future_years, 
                          interval = "confidence") |>
  as.data.frame() |>
  bind_cols(future_years)

ggplot() +
  geom_ribbon(data = predictions,
              aes(x = year, ymin = lwr, ymax = upr),
              alpha = 0.2, fill = "#E63946") +
  geom_line(data = enteritidis_trend,
            aes(x = year, y = pct_fluoroquinol),
            linewidth = 1.2, color = "#E63946") +
  geom_point(data = enteritidis_trend,
             aes(x = year, y = pct_fluoroquinol),
             size = 2.5, color = "#E63946") +
  geom_line(data = predictions,
            aes(x = year, y = fit),
            linewidth = 1.2, linetype = "dashed", color = "#E63946") +
  geom_vline(xintercept = 2025.5, linetype = "dotted", color = "gray40") +
  annotate("text", x = 2025.7, y = 5, label = "Projected →",
           color = "gray40", size = 3.5, hjust = 0) +
  scale_x_continuous(breaks = 2015:2030) +
  scale_y_continuous(limits = c(0, 100)) +
  labs(
    title = "Fluoroquinolone Resistance Trajectory in S. Enteritidis (2015–2030)",
    subtitle = "Linear projection based on 2015–2025 trend | shaded area = 95% confidence interval",
    x = "Year", y = "% Resistant Isolates",
    caption = "Source: NCBI Pathogen Detection Database | n = 57,297 clinical isolates"
  ) +
  theme_minimal(base_size = 13) +
  theme(plot.title = element_text(face = "bold"),
        panel.grid.minor = element_blank())

A linear model fit to the 2015–2025 S. Enteritidis trend (R² = 0.77) projects resistance reaching approximately 65–70% by 2030 if current trends continue. This projection assumes no major policy interventions, changes in antibiotic prescribing, or surveillance methodology shifts. The observed dip in 2020–2021 likely reflects COVID-19 disruptions to both prescribing patterns and surveillance submissions rather than genuine resistance decline.

Discussion

This analysis identifies fluoroquinolone resistance in S. Enteritidis as the most pressing trend in US clinical Salmonella surveillance over the past decade. Several limitations warrant consideration:

  • Geographic bias: NCBI submission patterns are uneven; states with more represented data likely reflect lab infrastructure rather than true epidemiological differences
  • Genotypic vs. phenotypic resistance: AMR genotypes predict but do not confirm phenotypic resistance — some genes may be non-expressed
  • Sample representativeness: This analysis uses a 10% random sample of available isolates; full dataset analysis may refine estimates
  • Linear projection caveats: Resistance trajectories are rarely linear indefinitely; projections should be interpreted as scenarios rather than forecasts
